# Personal Finance Manager

This project is a desktop application built with Python and FreeSimpleGUI.

## Features
- Add income and expenses
- Create and edit categories with custom colors
- Filter transactions by date
- Export data to CSV
- Data persistence using JSON files
- Unit tests included

## How to run the application

```bash
python3 main.py
```

## How to run tests

```bash
python3 -m unittest test_logic.py
```
